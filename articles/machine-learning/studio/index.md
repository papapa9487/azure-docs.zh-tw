# <a name="azure-machine-learning-studio-documentation"></a>Azure Machine Learning Studio 文件

Azure Machine Learning Studio 是共同作業式的拖放工具，您可以用來依據您的資料建置、測試及部署預測分析解決方案。
教學課程、影片和模型範例會示範如何使用 Studio 來建置和部署機器學習模型。
